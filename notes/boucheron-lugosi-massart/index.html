Concentration Inequalities : A Nonasymptotic Theory of Independence

Stéphane Boucheron, Gábor Lugosi, Pascal Massart

Basic Inequalities

2.1 [08/17/2018]

Let $\bfM Z$ be a median of the square-integrable random variable $Z$ (i.e. $\bfP\{Z\geq\bfM Z\}\geq 1/2$ and $\bfP\{Z\leq\bfM Z\}\geq 1/2$). Show that \[ \abs{\bfM Z-\bfE Z}\leq\sqrt{\Var(Z)} \]

Solution

Recall that $\bfM Z$ minimizes $\bfE\abs{Z-c}$ over $c\in\mathbb R$. Thus, \[ \abs{\bfE Z-\bfM Z} = \abs{\bfE(Z-\bfM Z)}\leq \bfE\abs{Z-\bfM Z}\leq \bfE\abs{Z-\bfE Z}\leq \bracks{\bfE\parens{\abs{Z-\bfE Z}^2}}^{1/2} = \sqrt{\Var(Z)}. \]

2.2 [08/18/2018]

Let $X$ be random variable with median $\bfM X$ such that positive constants $a$ and $b$ exist so that for all $t>0$, \[ \bfP\braces{\abs{X-\bfM X}>t}\leq ae^{-t^2/b}. \] Show that $\abs{\bfM X-\bfE X}\leq \min\parens{\sqrt{ab},a\sqrt{b\pi}/2}$.

Solution

The first bound is given by \begin{align*} \abs{\bfE X-\bfM X} &= \sqrt{\parens{\bfE\parens{X-\bfM X}}^2}\leq \sqrt{\bfE\parens{\abs{X-\bfM X}^2}} = \sqrt{\int_0^\infty \bfP\braces{\abs{X-\bfM X}^2>t}~dt} \\ &= \sqrt{\int_0^\infty \bfP\braces{\abs{X-\bfM X}>\sqrt t}~dt}\leq \sqrt{\int_0^\infty ae^{-t/b}~dt} = \sqrt{ab} \end{align*} and the second bound is given by \begin{align*} \abs{\bfE X-\bfM X} &= \abs{\bfE(X-\bfM X)}\leq \bfE\abs{X-\bfM X} \\ &= \int_0^\infty\bfP\{\abs{X-\bfM X} > t\}~dt\leq \int_0^\infty ae^{-t^2/b}~dt = a\sqrt{b\pi}/2. \end{align*}

2.3 (Chebyshev-Cantelli Inequality) [08/19/2018]

Prove the following one-sided improvement of Chebyshev's inequality: for any real-valued random variable $Y$ and $t>0$, \[ \bfP\braces{Y-\bfE Y\geq t}\leq\frac{\Var(Y)}{\Var(Y)+t^2}. \]

Solution

For every $\lambda\in\mathbb R$, we have the bound \[ \bfP\braces{Y-\bfE Y\geq t} = \bfP\braces{(Y-\bfE Y+\lambda)^2\geq (\lambda+t)^2}\leq \frac{\Var(Y) + \lambda^2}{(\lambda + t)^2}. \] Then to optimize the RHS with respect to $\lambda$, we find the critical point \[ \frac{(\lambda+t)^22\lambda - (\Var(Y)+\lambda^2)2(\lambda+t)}{(\lambda+t)^4} = 0\iff \lambda = \frac{\Var(Y)}{t} \] which gives the upper bound of \[ \frac{\Var(Y) + (\Var(Y)/t)^2}{(\Var(Y)/t + t)^2} = \frac{t^2\Var(Y) + \Var(Y)^2}{(\Var(Y) + t^2)^2} \]

2.4 (Paley-Zygmund Inequality) [08/20/2018]

Show that if $Y$ is a nonnegative random variable, then for any $a\in(0,1)$, \[ \bfP\braces{Y\geq a\bfE Y}\geq (1-a)^2\frac{(\bfE Y)^2}{\bfE[Y^2]}. \]

Solution

We have that \[ \bfE Y = \bfE\parens{Y\bbone_{\{Y\leq a\bfE Y\}}} + \bfE\parens{Y\bbone_{\{Y> a\bfE Y\}}}\leq a\bfE Y + \sqrt{\bfE Y^2}\sqrt{\bfP(Y>a\bfE Y)} \] so rearranging yields \[ \frac{(1-a)\bfE Y}{\sqrt{\bfE Y^2}}\leq \sqrt{\bfP(Y>a\bfE Y)}\iff \bfP(Y>a\bfE Y)\geq (1-a)^2\frac{(\bfE Y)^2}{\bfE Y^2}. \]
